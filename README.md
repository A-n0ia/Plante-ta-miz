# Plante-ta-miz

Plantamiz est un projet étudiant visant à recréer une sorte de "Candy Crush" dans le terminal.

=== REGLES DU JEU ===

          - OBJECTIF -
          
    Alignez des items identiques pour remplir le contrat.

          - ITEMS -
          
      S : Soleil
      F : Fraise
      P : Pomme
      O : Oignon
      M : Mandarine

          - COMMANDES -
          
    Deplacer le curseur : 
        - z (haut)
        - q 
        - s (bas)
        - d (droite)
        - FLECHES DIRECTIONNELLE POSSIBLE 
    Selectionner un item : barre d'espace
    Une fois un item sélectionné, vous pouvez le permuter avec un item adjacent en utilisant les commandes de déplacement 

          - REGLES DE SUPPRESSION ET POINTS -
          
      Groupe de 3 ou plus : disparition (activé à partir du level 2)
    Groupe de 4 : 4 points
    Groupe en H : 2*X points (X = nb items)
    Rectangle L x l : 2*(L*l) points
    Groupe de 6 : disparition de tous les items identiques

          - FIN DE PARTIE -
          
    5 vies max
    Sauvegarde du joueur et niveaux
    Possibilitée de recommencer ou de quitter
